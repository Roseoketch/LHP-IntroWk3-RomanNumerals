## by [Karen Freeman-Smith](http://karenfreemansmith.github.io)

### Description
__*A website that translates numbers into Roman-numerals.*__

#### Tests:
* 5    | "V"
* 16   | "XVI"
* 22   | "XXII"
* 777  | "DCCLXXVII"
* 333  | "CCCXXXIII"
* 1888 | "MDCCCLXXXVIII"
* 2444 | "MMCDXLIV"
* 3999 | "MMMCMXCIX"
* 4000 | error message
* non-numeric input is rejected - error message/try again
